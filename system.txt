VERIFY
LOCAL 0x00            * tim - Timer task
*
* Optional modules running on the host:
*
LOCAL 0xcf            * s7_mgt - Management/config task
LOCAL 0xef            * s7_log - Display and logging utility  
LOCAL 0x3d            * module id for camelgw core application
LOCAL 0x5d	      * module id for status_checker application (run at once type app)
LOCAL 0x6d	      * dummy module, we will use this module for link between gctload and other world, for configureing, for collecting event data and other
LOCAL 0x2d            * dsistat
LOCAL 0xed	      * user application: sniffer module - tcap module sent to this module trace messages (sniffa - daemon type app)
*
* ss7 stack modules running on the host:
*
LOCAL 0xd1            * SCTP native
LOCAL 0xd2            * M3UA module
LOCAL 0x33            * SCCP module
LOCAL 0x14            * TCAP module
LOCAL 0x35            * INAP module

* For SIGTRAN systems using SCTPN the following REDIRECT is required:
REDIRECT 0xd0 0xd1

* redirect MTP3 to M3UA
REDIRECT 0x22 0xd2
REDIRECT 0xc2 0xd2

* Redirection of status indications:
*
REDIRECT        0xdf    0x6d    * LIU/MTP2 status messages -> s7_log
*
DEFAULT_MODULE  0x6d            * Redirect messages by default to module 0x6d
*
* Dimensioning the Message Passing Environment:
*
NUM_MSGS        65000
NUM_LMSGS      1000
*
* Now start-up all local tasks:
*
FORK_PROCESS ./tim
FORK_PROCESS ./tick
FORK_PROCESS ./s7_mgt -d -kconfig.txt
FORK_PROCESS ./s7_log -n20 -td -s1000 -fcamelgw.log
FORK_PROCESS ./sctpn
FORK_PROCESS ./HSTBIN/m3ua -Lp/opt/DSI -d
FORK_PROCESS ./HSTBIN/sccp -Lp/opt/DSI -d
FORK_PROCESS ./HSTBIN/tcap -Lp/opt/DSI -d
FORK_PROCESS ./HSTBIN/inap -Lp/opt/DSI -d
